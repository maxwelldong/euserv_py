# euserv

**德鸡无脑脚本**

```
  本脚本支持euserv自动登录并检查是否可以续期，如果到续期时间自动续期。支持配置多账号，且多线程优化执行效率

  配置说明：
    EUSERV_EMAIL：配置euserv登录邮箱
    EUSERV_PASSWORD：配置euserv登录密码
    EMAIL_PASS：配置对应账号邮箱的应用专用密码（注意：这个密码需要去邮箱设置里面开启IMAP并生成应用专用密码，设置方法可以询问AI）
	TG_BOT_TOKEN：配置tg账号的token，非必须，不想收通知可以不配置
	TG_CHAT_ID：配置tg账号的userid，非必须，不想收通知可以不配置
	
```
